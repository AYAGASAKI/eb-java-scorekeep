Create a user

Enter a name and password to sign up for Scorekeep. Login information and a copy of the Scorekeep user id are stored in a Cognito User Pool. When you sign in with these credentials the app retrieves the Scorekeep user ID and shows the user information for the logged in user.

Log out to end the Cognito session and unload the Scorekeep user from scope. Log in and click Delete me to delete both Scorekeep and Cognito users.

All of the Cognito related code for this page is in cognitoController.js. See the changes (relative to the master branch) to mainController.js and userController.js for an object oriented example.
